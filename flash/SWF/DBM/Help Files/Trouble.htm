Troubleshooting

When you select a file in step 1, the image should appear above the word Preview. If nothing happens you probally don't have Javascript enabled. Please consult your browsers help file to enable Javascript or check out this site.

If you are using Safari, Opera, or Mozilla 1.0 (not Firefox) you will most likely encounter problems due to the way these browsers handle cookies.

If you still can't get Deviant Bordermaker working then I reccomend downloading Firefox (which you should be using anyway).

Make sure DBM.htm, help.htm, preview.htm, and HEXCode.htm are all in the same folder.

Installing the Deviant Bordermaker.js script for use in Photoshop.

  1. You must have Photoshop CS2/CS or Photoshop 7 with the Scripting Plugin.Mac / Windows
  2. Copy Deviant Bordermaker.js to your Photoshop Script's folder.
    <Photoshop folder*>\Presets\Scripts
    * Default for CS2 in Windows is C:\Program Files\Adobe\Adobe Photoshop CS2
  3. Restart Photoshop if it was running while we did that.
  4. In Photoshop, go to File -> Scripts and you should see DBM in the list.
