Walkthrough

Deviant Bordermaker is self explanatory but here is a break down of the program step by step.


If your image doesn't show up right away, click the "Get Image Properties" button.

 

  • Click the button and navigate to the image you wish to use.
  • Any image format that is supported by the browser will work.(.jpg, .gif, .png, etc.)
  • If you select a file type that the browser doesn't recognize nothing will happen.
  • Some browsers may require you to hit the button in order to load the image. (Specifically Safari, Opera, and Mozilla 1.0)

[Top]

  • By default all of the boxes are checked, so every aspect ratio will be calculated.
  • Deviant Bordermaker 2.0 remembers which boxes you had checked the last time you run the program.
  • Simply uncheck the ones you don't want to calculate or press the button.
  • Any unChecked ratios will be hidden in the results (See Step 4)

[Top]

  • This is the amount of space between the picture and the frame. It is measured in pixels but if you want to think in inches just remember 1 inch = 72 pixels.
  • Once you enter a number click anywhere else on the screen and the numbers in Step 4 will adjust automatically.
  • You can also select the border color via the drop down list.
  • Note: Deviant Bordermaker refers to the box next to the drop down list when applying the color to the border.
  • The drop down list is just a quick short-cut to the basic Black, White, and Gray border colors.
  • If you select "My Own Color..." from the drop down list, the box will say "Change" and automatically highlight so all you have to do is enter in your own Hex code.
  • Don't know what the hex code to your favorite color is? Click the "Color Codes" link and a new window will pop up with any color you could ever want with an example of that color.
  • Simply highlight the 6 letter/number combination and copy it to the box next to the drop down list in Deviant Bordermaker.

In this example, the missing ratios are not displayed because they were not checked in Step 2.

  • If the fields are empty click the button.
  • If you still see nothing scroll up to Step 2 and make sure atleast one aspect ratio is checked.
  • To make things easier for you, only the ratios you have checked will be visible.
  • Click the button to see what the border looks like around the picture(launches new window).
  • Note: If either the height or width are smaller than the original image height or width you will not see the image being clipped. Click here to see what I mean.
  • In the preview window you will notice that the image is scaled to fit inside the browser window.
  • If you change the dimensions of the browser window and hit refresh it should update accordingly.

[Top]

Here is where the output code will be produced and displayed.

  1. Select all of the code using the button.
  2. Copy (control + C)
  3. Open Photoshop CS/CS2 or Photoshop 7 with the scripting plugin.
  4. Run the Deviant Bordermaker.js script by clicking File->Scripts. If you do not see it in the list, click Browse and find it on your hard drive.
  5. Paste the code into the box and press the button.

 

The Preview results (left) with all the various info displayed at the bottom(above). You can change the color of the background color using the drop down menu. Deviant Bordermaker 2.0 will remember which color you pick.

And that is all there is to it!
