Firebug Firefox Edit, debug, and monitor CSS, HTML, and JavaScript live in any web page. All of them
Web Developer Toolbar Firefox Plethora of little web developer tools. Display Element Information (shows calculated font-size in pixels)

Display CSS by media type (great for debugging print stylesheets)
Dust-Me Selectors Firefox Finds unused selectors in your stylesheets as you browse through your site. Spider an entire site for automated checking
Yahoo YSlow Firefox Analyzes pages and suggests performance improvements based on a set of rules for high performance web pages. Breakout of page components and file sizes

Neat graphs

(#f = #fff = #ffffff)
DigitalColor Meter Mac Built in color picker that supports hex values. Zoom-in picker

Adjustable aperture for averaging
